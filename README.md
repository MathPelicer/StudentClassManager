# StudentClassManager

### 1. Base de dados

- Rodar script da base de dados `scripts/script.sql`

### 2. Executar aplicação:

#### Comandos:

`dotnet run src/StudentClassManager.Api`

`dotnet run src/StudentClassManager.UI`
